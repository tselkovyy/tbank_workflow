# Описание
Каждый день в 9 утра (МСК) через чат бота мне отправляются вакансии с сайта hh.ru по указанному направлению указанным грейдам. Мною было принято решение искать вакансии в том числе и на русском языке, это же касается и грейдов (также учтены регистр и буквы е/ё в слове "стаж(е/ё)р"). Столкнулся с каким-то багом: после выполнения процесса при просмотре "внутренностей фильтров" слетает параметр caseSensitive. Подходящие вакансии отправляются через бота мне в телеграм, если подходящих вакансий не будет, то бот известит меня об этом.
